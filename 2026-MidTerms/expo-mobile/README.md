# Know Your Reps Mobile

This folder contains an Expo prototype for the mobile-first brutalist version of KNOW YOUR REPS.

## Run locally

```bash
cd expo-mobile
npm install
npx expo start
```

## API target

The app expects the web API to be available at:

- `https://two026-midterms.onrender.com`

Override it with:

```bash
EXPO_PUBLIC_API_BASE_URL=https://your-api-url npx expo start
```
